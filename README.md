# Argaklus Netlify Deploy

Deze map is klaar om naar GitHub te uploaden en via Netlify te deployen.

Belangrijk:
- Publiceer de root van deze map.
- Formulieren zijn voorbereid voor Netlify Forms.
- PHP-bestanden zijn bewust niet meegenomen, omdat Netlify statische hosting gebruikt.

Lokale preview:
```powershell
cd C:\Users\User\Desktop\argaklus-netlify-deploy
python -m http.server 5500
```
Daarna openen: http://localhost:5500
